# ANYCOLOR 專案基本架構

## 快速開始

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

開啟瀏覽器到 <http://127.0.0.1:5000/>。

## 專案結構

```text
.
├─ app.py
├─ events.json
├─ requirements.txt
├─ templates/
│  └─ index.html
└─ static/
   ├─ script.js
   └─ style.css
```
